\doxysubsection{Detailed Description}
A simple demo class to implement the operation of an abstract biological neuron. The operating principle of neuronal processing, and programming details\+: 

The class adjusts member sc\+Gen\+Comp\+\_\+\+PU\+\_\+\+Bio\+::m\+\_\+\+Membrane\+\_\+V (the membrane potential). Actually, it implements the \doxyref{AP}{p.}{GLOSSARY_GLOSSARY_AP} creation as shown in Figure \doxyref{fig\+\_\+\+AP\+\_\+\+Conceptual}{p.}{group__MODELING__SINGLE__AP_fig_AP_Conceptual}.

Given that the membrane\textquotesingle{}s potential rather than time of instruction count defines processing phases, method sc\+Gen\+Comp\+\_\+\+PU\+\_\+\+Abstract\+::\+Heartbeat\+\_\+method() is called periodically, with HEARTBEAT\+\_\+\+TIME\+\_\+\+DEFAULT (simulated) time (presently set to $10\ \mu s$). That function internally calls one of its state-\/dependent Heartbeat\+\_\+\+XXX\+\_\+\+Do() methods.

\doxyref{Input\+Received\+\_\+\+Do()}{p.}{classNeuronDemo_a6b1fa359d1a0d044e2d53e688af5a28f} imitates chargeup\+: membranes voltage increases promptly by $6\ mV$ after receiving a synaptic input

\doxyref{Heartbeat\+\_\+\+Computing\+\_\+\+Do()}{p.}{classNeuronDemo_a968f505fae12f7408930e79b117e61d2} imitates membrane\textquotesingle{}s discharge by decreasing its voltage by $1\ mV$

\doxyref{Heartbeat\+\_\+\+Delivering\+\_\+\+Do()}{p.}{classNeuronDemo_a7b536da8e59b248fb2c9b8cd7ec6cfa7} imitates creating a sawtooth-\/like \doxyref{AP}{p.}{GLOSSARY_GLOSSARY_AP} on the membrane

\doxyref{Heartbeat\+\_\+\+Relaxing\+\_\+\+Do()}{p.}{classNeuronDemo_a233a6008741bddf772d89a6211854329} imitates restoring potential to Resting\+Membrane\+Potential by changing its value by by $\pm 1\ mV$

The states of the neuronal state machine are controlled by methods Heartbeat\+\_\+\+XXX\+\_\+\+Stop()

\doxyref{Heartbeat\+\_\+\+Computing\+\_\+\+Stop()}{p.}{classNeuronDemo_aab4c43c084329cbc1de7975eed5c992f} passes to phase \textquotesingle{}Delivering\textquotesingle{} if membrane\textquotesingle{}s voltage exceeds the Threshold\+Potential or passes to phase \textquotesingle{}Relaxing\textquotesingle{} if it falls below Resting\+Membrane\+Potential (calls sc\+Gen\+Comp\+\_\+\+PU\+\_\+\+Abstract\+::\+Failed\+\_\+method() )

\doxyref{Heartbeat\+\_\+\+Delivering\+\_\+\+Stop()}{p.}{classNeuronDemo_a679dac13f103ce4140dc46b058f2fb82} passes to phase \char`\"{}\+Relaxing\char`\"{}, practically at a constant time. It increases membrane\textquotesingle{}s voltage until $\approx 150\ mV$ reached, then decreases in somewhat lower pace until Threshold\+Potential reached (imitate hyperpolarization)

\doxyref{Heartbeat\+\_\+\+Relaxing\+\_\+\+Stop()}{p.}{classNeuronDemo_a2d7f92be89e33e11e63bcab6f7e0760d} decreases membrane potential until -\/Threshold\+Potential reached, then restores Resting\+Membrane\+Potential reached

The class needs (empty) constructor and destructor. \doxyref{Initialize\+\_\+\+Do()}{p.}{classNeuronDemo_ad115de52a3a36b827e4f5cbc639b2016} is only called in constructor (preserves voltage between spikes)
