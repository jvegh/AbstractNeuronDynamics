\doxysubsection{Detailed Description}
Implements a general biological-\/type computing PU. Defaults to variable execution time, no central clock. 

In the corresponding states, the neuron is in one of its Gen\+Comp\+State\+Machine\+Type\+\_\+t states
\begin{DoxyItemize}
\item gcsm\+\_\+\+Computing\+: computing (collects charge)
\item gcsm\+\_\+\+Delivering\+: delivers action potential
\item gcsm\+\_\+\+Relaxing\+: restores resting potential
\end{DoxyItemize}

The Heartbeat\+\_\+method() handles an internal \textquotesingle{}update\textquotesingle{} signal. In the corresponding states, the neuron is
\begin{DoxyItemize}
\item gcsm\+\_\+\+Relaxing\+: Until it reaches its resting potential, calls Calculate\+Membrane\+Potential(). Upon receiving an input, changes to state gcsm\+\_\+\+Computing. Inputs enabled\+: the relative refractory period.
\item gcsm\+\_\+\+Computing\+: Until it reaches its threshold potential, calls Calculate\+Membrane\+Potential(). At that point, changes to state to state gcsm\+\_\+\+Delivering. Inputs enabled.
\item gcsm\+\_\+\+Delivering\+: Until it reaches its lowest membrane potential (its increase turns from negative to positive), calls Calculate\+Membrane\+Potential(). At that point, changes to state to state gcsm\+\_\+\+Relaxing, without restoring membrane\textquotesingle{}s potential. At the beginning issues signal \textquotesingle{}Begin Transmitting\textquotesingle{}. Signal transmission happens in parallel with the further processing. Inputs disabled; the absolute refractory period.
\end{DoxyItemize}

The Input\+Received\+\_\+method()\+: the unit received new input (a momentary state). Only administrative action; the received input is handled in Heartbeat\+\_\+method(), as described in Calculate\+Membrane\+Potential(). Disabled in mode gcsm\+\_\+\+Delivering.
